Claude Code é para quem vive no terminal. Cowork é para quem não precisa — e nunca quis. Ambos compartilham o mesmo motor agentic. Esta página cobre instalação, primeiros comandos e quando usar cada um.

Terminal-first, para devs

Acessa seu codebase, executa comandos, reescreve arquivos e faz git — tudo por linguagem natural. Você dirige, Claude executa em loop até concluir.

Desktop-first, para todos

Aponta para uma pasta, descreve o objetivo. Claude planeja e executa tarefas multi-etapa — análise, organização, relatórios — sem tocar no terminal.

1
Instalação
macOS / Linux
# Instalador nativo — não precisa de Node.js
$ curl -fsSL https://claude.ai/install.sh | sh
✔ Claude Code instalado com sucesso

$ claude --version
Claude Code 2.x.x_

Sem Node.js. O instalador nativo é standalone. Se você usava o método npm, pode migrar — suas configurações em ~/.claude/ são preservadas.

Node.js 18+
# Requer Node.js 18+
$ node --version
v20.11.0

$ npm install -g @anthropic-ai/claude-code
PowerShell / CMD
# Opção 1: WinGet
PS> winget install Anthropic.ClaudeCode

# Opção 2: WSL2 (projetos Linux)
PS> wsl --install
↳ depois use o instalador nativo dentro do WSL

Dica Windows: use Git for Windows no modo nativo. Para projetos em WSL2, execute o Claude Code dentro do próprio WSL — evita latência de I/O cross-filesystem.

3
CLAUDE.md — contexto persistente

Crie um arquivo CLAUDE.md na raiz do projeto. É carregado em cada sessão automaticamente. Pense nele como o briefing do estagiário antes de começar.

CLAUDE.md — exemplo mínimo
# Projeto: Landing Page Produto X

## Stack
- Next.js 14 (App Router), TypeScript, Tailwind
- Banco: PostgreSQL via Prisma

## Regras
- Não modifique prisma/migrations/ diretamente
- Sempre rode pnpm lint antes de concluir
- Preferência: named exports, sem classes

## Contexto de negócio
- Produto de B2B SaaS, foco em PMEs
- Tom: direto, sem jargão excessivo

Dica prática: mantenha o CLAUDE.md enxuto. Linke para arquivos externos se necessário — ele é carregado em toda sessão e consome tokens de contexto.

4
Plan Mode — para tarefas complexas
01

Ative antes de implementar

Use Shift+Tab para ativar Plan Mode. Claude mostra o plano antes de executar — ideal para tarefas que tocam 3+ arquivos.

02

Revise o plano antes de aprovar

O Claude lista os arquivos que vai modificar e a sequência de operações. Você confirma ou ajusta antes de qualquer escrita.

03

Push back é esperado

Se o approach não for o que você quer, diga explicitamente: "Não, use X em vez disso". Claude itera sem rancor — foi treinado para adaptar.

O que é e como acessar
  • ↓
    Baixe o Claude Desktop App
    Disponível para macOS e Windows. Acesse claude.ai/download — é o mesmo app do chat, agora com uma aba Cowork na sidebar.
  • ⊞
    Abra a aba "Cowork"
    Aparece na sidebar ao lado do Chat. Se não aparecer, atualize o app para a versão mais recente.
  • 📁
    Monte uma pasta
    Claude só acessa o que você autorizar. Comece com uma pasta de teste — não aponte para a raiz do sistema logo de cara.
  • ✓
    Descreva o objetivo
    Fale o resultado que quer, não os passos. "Organize esses PDFs por mês e gere um índice em Markdown." Claude planeja e executa.

Research Preview: Cowork ainda está em fase de preview. Disponível para todos os planos pagos (Pro, Max, Team, Enterprise). Free não tem acesso.

2
Como Cowork funciona na prática
01

Você define o objetivo

Fale em linguagem natural o que precisa. Cowork é orientado a outcome — não a prompt.

02

Claude planeja as etapas

Antes de agir, exibe o plano. Você pode aprovar, ajustar ou cancelar. Deletar/editar arquivos pede confirmação explícita.

03

Execução com acompanhamento em tempo real

Você vê as tool calls acontecendo: leitura de arquivo, escrita, chamada de conector. Pode pausar a qualquer momento.

04

Resultado entregue na pasta

Arquivos transformados ficam onde você autorizou. Cowork não sai da sandbox que você definiu.

Casos de uso — não-técnicos
  • 📊
    Análise de planilhas
    Exporta para xlsx docx pdf com skills nativas. Ex: analisa despesas do mês e gera relatório Word formatado.
  • 📂
    Organização de arquivos
    Classifica, renomeia, move por padrão. "Agrupe os PDFs de notas fiscais de 2024 por trimestre."
  • 🔍
    Pesquisa e síntese
    Com connector Claude in Chrome, navega e compila informações de múltiplas páginas sem você trocar de janela.
  • 📋
    Relatórios multi-fonte
    Cruza dados de arquivos locais com conectores externos (Notion, Google Drive, n8n) e entrega um documento consolidado.
4
Conectores — expandindo o alcance

Cowork acessa serviços externos via MCP connectors. O catálogo tem centenas de opções — alguns destaques:

  • 🌐
    Claude in Chrome
    Extensão de browser. Permite que Cowork execute tarefas via web sem você trocar janelas.
  • 📁
    Google Drive / Notion
    Ainda em desenvolvimento para Cowork. Por enquanto, use o workaround via Chrome extension para workflows Google.
  • ⚙️
    n8n / AWS / Honeycomb
    Para automação de workflows, cloud e observabilidade. Conectores são revisados pela Anthropic antes de publicação.

Atenção de segurança: não use Cowork com dados sensíveis regulados (PII, HIPAA, FSI) na fase de research preview. Há vulnerabilidade conhecida de prompt injection em documentos — a Anthropic recomenda cautela com arquivos de fontes desconhecidas.

Primeiros prompts para testar

Cole estes diretamente — são pontos de partida calibrados, não exemplos genéricos.

Claude Code · Setup

"Leia o projeto nessa pasta e me explique a arquitetura em 5 pontos. Depois pergunte o que eu quero melhorar."

Claude Code · Refactor

"Entre em Plan Mode. Identifique as 3 funções mais complexas do projeto e proponha um refactor para cada uma antes de executar."

Claude Code · Debug

"Rode os testes. Para cada falha, explique a causa raiz antes de corrigir. Não mude o comportamento de testes que passam."

Claude Code · Git

"Crie um branch feat/[nome], implemente X, rode o lint, e gere um commit com mensagem no padrão conventional commits."

Cowork · Organização

"Nessa pasta há PDFs misturados de anos diferentes. Crie subpastas por ano/mês e mova os arquivos. Mostre o plano antes de executar."

Cowork · Análise

"Analise essa planilha de despesas e gere um relatório Word com: total por categoria, 3 insights principais e um gráfico de barras."

Cowork · Pesquisa

"Compile os últimos 5 relatórios desta pasta num único PDF com sumário executivo de 1 página no início."

Cowork · Síntese

"Leia todos os arquivos .txt nesta pasta e produza um documento Markdown estruturado com os pontos mais recorrentes entre eles."
